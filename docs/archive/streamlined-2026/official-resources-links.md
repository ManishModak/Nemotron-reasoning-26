# Official Resources & Links

A comprehensive list of NVIDIA and community resources for the Nemotron Model Reasoning Challenge.

## Core Nemotron Model Family
- **Nemotron GitHub**: Main repo and examples [Link]
- **Nemotron v3 Collection**: Hugging Face ready-to-use checkpoints [Link]
- **Nemotron 3 Nano Technical Blog**: Details architecture, efficiency, and training specifics.

## NeMo Open Libraries
- **NeMo DataDesigner** (Synthetic data): GitHub & Docs available for generating task-specific data.
- **NeMo Curator** (Data filtering): Multi-modal data curation toolkit GitHub & Docs.
- **NeMo RL** (RL training): At-scale training with SOTA algorithms (e.g., PPO, GRPO).
- **NeMo Gym** (RL environments): Contains 100+ Reasoning Gym environments.

## Open Datasets
NVIDIA has released multiple datasets to assist with post-training and alignment:
- Pre-training and Post-training dataset collections
- RL instruction-following dataset
- Agentic dataset
- RAG dataset
- Personas collection

## Guides & Tutorials
- **Nemotron 3 Super**: Multi-stage RLVR & RLHF recipe using NeMo RL.
- **GRPO RLVR Jupyter Notebook**: Step-by-step notebook in the Nemotron usage-cookbook.
- **Nemotron 3 Nano RL Guide**: Nightly guide for exact Nano RL reproductions.
- **Nemotron 3 Nano Reproducibility/Eval Guide**: Setup with NeMo Evaluator.

## Community & Learning Paths
- **Discord**: NVIDIA Developer Discord (Nemotron channel) and the Reasoning Challenge specific channel.
- **Email**: community@nvidia.com for questions and sharing work.
- **Learning Paths**: Guides on "How to build an AI agent" and "How to build a RAG agent".
- **Launchable Environments**: Pre-configured cloud environment via brev.dev.
- **Livestreams**: Nemotron Labs livestreams playlist on YouTube (deep dives, demos, Q&A).
