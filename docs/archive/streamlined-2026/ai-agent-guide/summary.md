# Agentic AI Foundations Learning Path

**Title:** Agentic AI Foundations Learning Path

This is a central hub on the NVIDIA Developer portal that outlines the foundational technologies and learning paths required to build sophisticated AI agents.

*   **Key Information:**
    *   **Focus Areas:** Covers a broad spectrum of AI topics including Generative AI, Retrieval-Augmented Generation (RAG), Conversational AI, and Vision AI.
    *   **Platform Integration:** Connects developers to NVIDIA tools like the CUDA Toolkit, NIM (NVIDIA Inference Microservices), and the NGC Catalog for GPU-optimized software.
    *   **Industry Applications:** Tailors agentic AI concepts for sectors such as Robotics, Autonomous Vehicles, Financial Services, and Healthcare.
    *   **Community Resources:** Provides links to the NVIDIA Developer Discord, technical blogs, and GTC (GPU Technology Conference) sessions for deeper learning.

**Source:** [How to Build an AI Agent](https://developer.nvidia.com/topics/ai/how-to-build-an-ai-agent)
